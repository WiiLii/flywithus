### The Survey Branch

Built on the XAMPP Stack.

* Apache - Server Stuff
* mySQL - Database Stuff

### Setup
1) Run XAMPP
2) Add CF1 to htdocs
3) Create database 'flywithus'
4) Execute runMe.sql
5) Run Apache & mySQL servers
6) Go to localhost/CF1 in browser
